# Extreme-Hand-Tracking-FX
Real time computer vision engine for hand tracking and interactive AR visual effects.

An advanced, real-time Augmented Reality (AR) engine for the browser. This project leverages Google's **MediaPipe** neural network to track hand joints in 3D space and render dynamic, reactive visual effects.

## Features
- **Dynamic Polygon Masking:** Real-time AR masking that fuses hand movements into interactive polygons.
- **Gesture Recognition:** Includes support for Gestures (Fist, Pinch, Point, Open) to trigger different visual modes.
- **High-Performance FX:** Includes unique modules like *Scanner, X-Ray, Thermal, NodeGrid, Symbiote, and Overcharge.*
- **Optimized Rendering:** Engineered for high frame rates using custom canvas drawing protocols.

## How to use
1. Visit the live site: https://chittaranjankakati.github.io/Extreme-Hand-Tracking-FX/
2. Click **"Engage Systems"** to boot the tracking engine.
3. Grant camera permissions when prompted.
4. Interact using your hands and experiment with different gesture-based modes.

## Tech Stack
- **AI/ML:** [MediaPipe Hands](https://mediapipe.dev/)
- **Graphics:** HTML5 Canvas API
- **Language:** Vanilla JavaScript / HTML5 / CSS3

## Troubleshooting
- If the camera doesn't start, ensure you are using a modern browser (Chrome/Edge/Safari) and that you've granted camera access.
- For optimal performance, ensure you are on a stable connection for the initial model load.
